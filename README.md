# ToDo  App 

ToDo App  Using Cubit as State management with Light and Dark Mode

# featured used in app :
- Clean Architecture Design Pattern
- MVVM - Model - View - View Model Pattern
- Dependency injection
- Application Layer
- Domain layer
- Data layer
- Presentation layer
